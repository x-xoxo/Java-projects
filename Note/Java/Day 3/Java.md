# Java

## 형변환(Casting)

- 기본적으로 기본형 자료형끼리 형변환 가능
- 기본적으로 형변환 안하는게 좋음

## 반복문

```java
while(b<10)  // 정해진 횟수 없이 어떤 조건에서 반복하고 싶을 때 사용
{
    System.out.println("Hello World!");
    b++;
}
```

```java
for(int a = 0; a < 10; a++) // 정해진 횟수만 큼 반복하고 싶을 때 사용
{
    System.out.println("Hello World!");
}
```

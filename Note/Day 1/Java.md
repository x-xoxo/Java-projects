## Operator(연산자)

단항 연산자: +a, -a, a++, a--, --a, ++a  
이항 연산자: a+b, a-b  
삼항 연산자:a > b ? a + 5 : a - 5  
대입 연산자: = 좌측에 대입  
ex)int a = 10; // 해석 : 10이 정수형 변수 a 에 대입 됐다.
증감 연산자  
int a = 10;  
a++;   // 전위형 a = a + 1  
++a;   // 후위형  
a--;   // a = a - 1  
--a;
